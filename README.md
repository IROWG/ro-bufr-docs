# RO BUFR Working Documents

This folder contains a collection of working documents originally downloaded from the older Google Drive on 17 April 2026. Going forward, these documents will be maintained here on GitHub rather than on Google Drive.

## Purpose

These documents support the development, evolution, and implementation of a new buffer format for radio occultation (RO) data.

## Context

This work is carried out under the auspices of [IROWG](https://www.irowg.org) (the International Radio Occultation Working Group), a scientific working group operating under the [CGMS](https://www.cgms-info.org) (Coordination Group for Meteorological Satellites) of the WMO (World Meteorological Organization).
